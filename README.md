# TZ-java-AQA
Требования:
  1)Java 21+

  2)Google Chrome (последняя версия)

  3)Maven 3.6+
  
-------------------------------------------
  Для Linux / macOS
Способ 1: Основной скрипт(через терминал)

      # Дать права на выполнение
      chmod +x run-tests.sh
      # Запустить с меню выбора
      ./run-tests.sh
      # Или запустить сразу тест 3
      ./run-tests.sh CreateWidgetTest
      
-------------------------------------------
  Для Windows
  
  1)Найти файл run-tests.bat в корне проекта

  2)Запустить двойным кликом

  3)Выбрать нужный тест из меню
